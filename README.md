# My Coffee Co.

Premium freshly roasted coffee e-commerce web application, built with React, Vite, Tailwind CSS v4, and Framer Motion.

## Features
- Dynamic, responsive UI with smooth animations
- Best-in-class performance using Vite & React 19
- Modern modular component architecture

## Development

To start the development server:
```bash
npm install
npm run dev
```

To build for production:
```bash
npm run build
```
